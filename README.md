# Booksy Clone

Simple booking platform for salons and clients built with Next.js, React Native and nano.tech database.

## Tech stack

- Next.js (web + API)
- React Native (mobile app)
- nano.tech (database / main storage)

## Panels overview

Client panel – used to create an account, log in, browse salons and services, and book, view, or cancel personal appointments.

Salon panel – allows owners and staff to manage services, schedules, and bookings, and to view the appointment calendar.

Admin panel – used to manage users and salons (activation/deactivation), monitor the quality of listings, and access basic system statistics.

## Functionality

Client – registration/login panel, searching for salons and services, creating appointments in available time slots, profile section with the list of appointments, managing appointments (reschedule/cancel) and basic account settings (avatar, bio, etc.).

Salon – creating and managing services (name, price, duration), managing appointments in the salon calendar, and basic salon profile settings (name, description, image, contact details, etc.).

Admin – an additional panel used to see all appointments and access key information about users and salons (for example basic details and status).
